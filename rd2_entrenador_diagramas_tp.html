Entrenador de diagramas tipo TP

Practicá lo que puede preguntar el profe mirando una topología: cuántos routers, switches, PCs, LANs, enlaces WAN, redes IP y gateways.

Regla rápida

PC → Switch → Router = una LAN.
Router ↔ Router = red punto a punto/WAN, suele ser /30.
Router = separa redes y hace de gateway.
Redes IP totales = LANs + enlaces router-router + Internet si aparece.
